Ejercicio

Demuestra que si $f \in C[a,b]$ y $f(a) f(b) < 0 $, entonces existe $c \in (a,b)$ tal que $f(c)=0.$

Ejercicio

Crea un programa utilizando Python o Scilab que implemente el Método de Horner para evaluar polinomios. El programa debe solicitar el grado $n$ del polinomio $p(x)$, el punto $c$ en el que se debe evaluar el polinomio y regresar $p(c)$.

Ejercicio

Usa el programa creado en el Ejercicio 1.2 para hallar $p(c)$ en los siguientes casos:

  1. $p(x)= x^4 + x^3 -13x^2 -x -12, c=3$
  2. $p(x)= 2x^7 + x^6 + x^5 -2^4 -x +23, c=-1$

Ejercicio

Crea un programa utilizando Python o Scilab que reciba un número real positivo $x$, y devuelva la correspondiente representación binaria de $x$, use $32$ bits para representar la parte entera y $32$ bits para representar la parte decimal. Por ejemplo, si $x=1.5$ el programa devolvera $$00000000000000000000000000000001.10000000000000000000000000000000$$

Sugerencia: Convierta la parte entera y la parte fraccionaria por separado.
